# 🔐 SecureVault Browser Extension

Transformez votre coffre-fort SecureVault en une expérience de navigation fluide grâce à notre extension professionnelle.

## 🚀 Installation (Mode Développeur)

### Pour Chrome, Brave, Edge, Opera, etc. :
1.  Ouvrez votre navigateur.
2.  Accédez à la page des extensions : `chrome://extensions`.
3.  Activez le **Mode développeur** (interrupteur en haut à droite).
4.  Cliquez sur **Charger l'extension décompressée** (Load unpacked).
5.  Sélectionnez le dossier `extension` situé dans le répertoire de votre projet.

### Pour Firefox :
1.  Accédez à `about:debugging#/runtime/this-firefox`.
2.  Cliquez sur **Charger un module complémentaire temporaire**.
3.  Sélectionnez le fichier `manifest.json` à l'intérieur du dossier `extension`.

## ✨ Fonctionnalités incluses

- **Accès rapide** : Consultez et copiez vos identifiants sans quitter votre onglet actuel.
- **Autocomplétion intelligente** : SecureVault détecte les champs de connexion et les marque avec le trait orange BELLOX pour confirmer la protection.
- **Sécurité Manifest V3** : Utilisation des dernières normes de sécurité imposées par Google.
- **Multisupport** Compatible avec tous les navigateurs modernes.

## 🛡️ Sécurité
L'extension communique directement avec votre backend local (`http://localhost:3001`). Assurez-vous que votre serveur Node.js est en cours d'exécution pour que l'extension puisse se synchroniser.
