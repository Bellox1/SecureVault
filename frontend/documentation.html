Premiers Pas

Création de compte

Pour commencer avec SecureVault, inscrivez-vous avec une adresse email valide. Un lien d'invitation sécurisé vous sera envoyé. Les liens expirent après 5 minutes pour garantir votre sécurité maximale.

Le Coffre-fort

Une fois connecté, vous accédez à votre tableau de bord. Vous pouvez y ajouter des identifiants (Login, Password, URL) de manière illimitée. Toutes les informations sont chiffrées localement avant stockage.

Coffre-fort & Sécurité

Chiffrement de bout en bout

Nous utilisons les algorithmes de chiffrement les plus robustes de l'industrie (AES-256) pour protéger vos données. Vos mots de passe ne sont jamais visibles, même par nos administrateurs. Seul votre mot de passe maître peut déverrouiller l'accès à vos informations.

Configuration 2FA

SecureVault supporte l'authentification à deux facteurs via Google Authenticator, Authy ou FreeOTP.

Activation

Rendez-vous dans l'onglet "Mon compte" de votre tableau de bord, scannez le QR Code généré et entrez le code secret à 6 chiffres pour lier votre appareil.
